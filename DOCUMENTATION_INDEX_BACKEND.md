## üì± Quick Command Reference

### Start the Backend
```bash
cd /Users/oyelade/paylink
npm install  # if needed
node server.js
```

### Test Live Providers Endpoint (No Auth)
```bash
curl http://localhost:5000/api/payflex-proxy/providers/airtime
```

### Make a Payment (With Firebase Token)
```bash
curl -X POST http://localhost:5000/api/payments/airtime \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "x-device-id: device_id" \
  -d '{"phone":"09012345678","amount":100,"provider":"MTN"}'
```

### Get Wallet Balance (With Firebase Token)
```bash
curl http://localhost:5000/api/wallet/balance \
  -H "Authorization: Bearer YOUR_TOKEN"
```
